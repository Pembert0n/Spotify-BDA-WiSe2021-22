# Spotify-BDA-2021-22
<br/>
Dieses Projekt wurde im Rahmen der Veranstaltung "Betriebswirtschaftliche Datenanalyse (282135)" an der Hochschule Heilbronn erstellt.<br/>
Vorgelegt bei Dr. Benjamin M. Abdel-Karim.<br/>
<br/>
Um dieses Projekt wie vorgesehen verwenden zu können müssen die Dataframes "Database to calculate popularity.csv" und "Final database.csv" runtergeladen werden.<br/>
Diese sind verfügbar unter https://www.kaggle.com/pepepython/spotify-huge-database-daily-charts-over-3-years.<br/>
Die beiden Dataframes müssen im selbern Verzeichnis wie die "Main.py" des Projekts gespeichert sein.<br/>
Darauf folgend müssen Sie umbenannt werden, so das Leerzeichen zu Unterstrich ersetzt werden.<br/>
Die Dataframes sollten dann folgende Namen haben: "Database_to_calculate_popularity.csv" und "Final_database.csv".<br/>
